All options on one 13-col Fibonacci grid · 8/5 splits · φ-descent rows
Grid system
13-column grid · gutter 13px · Hero row 600px → next rows in φ-descent (370 / 230 / 140). Horizontal splits use Fibonacci spans: 5 + 8 = 13. Internal subdivisions use 3 + 5 or 2 + 3. Card padding 13px outer, gutter 13px.
Aspect-ratio rules
Treemaps live only in slots with aspect 1:1 to 4:3 — c5×hero (≈1.18:1) or c5×row2 (≈1.4:1). Anywhere wider becomes a stacked bar in disguise. Linear charts (bars / stacked bars / time-series lines / lists) work fine in narrow bands — c8×row3, c5×row4, c3×row4. NS3451 codes-present + MMI distribution are stacked bars, not treemaps — they belong in linear slots. Discipline is a single-model field (lives in the Identity card, not as a chart).
