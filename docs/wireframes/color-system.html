docs/wireframes/color-system.html Color system — triangle gradient + signal orange Iterable mockup. Not bound to live code yet.

1 · Triangle gradient (lime → forest → navy)

Polyline through OKLCH color space, three vertices. Doubles the perceptual range of a single-segment gradient. The hue rotation at the forest vertex (155° → 260°) acts as the discipline cliff — classes on either side don't perceptually blur together.

Structure · 0.75 – 1.00 Architecture · 0.45 – 0.75 MEP · 0.20 – 0.45 Proxy / unknown · 0 – 0.20

2 · Canonical class positions

Each IFC class has a fixed position t ∈ [0,1] on the polyline. Same class → same color in every project, every reload. Position encodes discipline.

5 · Signal orange · cross-filter active states

One CSS variable --signal: oklch(0.78 0.18 50). Always means "this is what you picked". Lead filter (first key activated) carries the ◆ origin glyph. Secondary active filters share the orange outline but no glyph.
