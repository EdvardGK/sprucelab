Data source key — 7-tile KPI strip

  • Types analysis.total_types + orphan_types (rename → unused)
  • Classified typemapping_count / total_types against EIR-declared classification
  • With material types_with_material / total_types (new aggregation)
  • Untyped untyped_instances / total_products (today, in computeKpiStats)
  • Reuse Σ mapped_item_count / Σ representations
  • Proxy + Userdef proxy_count + userdefined_count (new aggregation)
  • Orphan total_products − Σ storey.element_count (shipped)

Context fields (Schema, MVD, Authoring tool, CRS, Basepoint, Units, File) are owned by the Metadata tab. The current ModelInfoCard at the bottom of Overview duplicates those — drop it as part of this PR so Overview is KPI-strip + Storeys + Elements + Quality + Geometry, no context echo.
